Off-Boarding Checklist

  • Provide mailbox access to
  • Provide OneDrive access to
  • Disable account in Active Directory
  • All security groups removed
  • Move user to Terminated OU in AD and specify term date in description field
  • Sign in Block in Admin Center
  • Remove the Cloud PC Licences

On-Boarding Checklist

  • Check Email and proxy address attribute are correctly set
  • Check the Preferred Data Location attribute is correctly set
  • Contractor moved to correct OU
  • Contractor added to Basis Default AD Groups?
  • Contractor is added to 3E Contractors Group?
  • Contractors assigned with Service Desk Self Service Portal Access?
  • Mimic the DLs / Security Groups as per Same As Account?
  • Check the OKTA Dashboard setup
  • Contractors assigned with Azure Cloud PC / Horizon Virtual PC?
  • Default applications / SW installed on PC?
  • Contractor Login Details Sent to Manager
  • Login instructions and OKTA setup instructions sent.
  • Ensure you add all the post check details in Ticket before closing.

    Kindest regards
    IT Tech Support,

    @username Haqani has requested a paid license for Lucidchart. This costs 365 USD per user per year. Please confirm approval for this request.

    Kindest regards
    IT Tech Support

Approval Process

  • Download the attachment
  • Save the attachment with format Requestor_surname_UAM
  • Open the saved attachment from location
  • Complete the information required
  • Open ‘IDM upload’ Outlook template from Mailbox
  • Attach completed UAM
  • Send email to EAAA IDM upload
  • Ensure the UAM is filled and saved to the correct naming convention
  • Update and close the service manager ticket.

OKTA Setup Mail

    Step 1: You will receive a SMS on your mobile to verify your account.
