Vulnerability Template

Subject: - Action Required: Update Google Chrome

Dear @username ,

It seems you might be running on an older Chrome version, which has some security vulnerabilities so it is important to keep Chrome up to date.

This should also be happening automatically so when you open Chrome check if at the top right it tells you there is a new version. Let us know if you see anything like this.

Furthermore, please follow these steps to your Google Chrome browser to the latest version by following these steps:

1. Open Chrome.
2. Click the three vertical dots (Menu) in the top-right corner.
3. Go to "Help" > "About Google Chrome".
4. Chrome will check for updates and install them.
5. Click "Relaunch" to complete the update.

Kind Regards,
IT Team
